# Online Store Sales Database (MySQL)

## Description
This project simulates a basic online store backend using MySQL.
It manages customers, products, and orders and performs real-world sales analysis using SQL queries.

## Database Used
MySQL

## Tables
- customers
- products
- orders

## SQL Concepts Used
- CREATE TABLE
- PRIMARY KEY & FOREIGN KEY
- INSERT
- JOIN
- GROUP BY
- HAVING
- Aggregate Functions

## Sample Queries
- Total sales per customer
- Top selling products
- Monthly sales analysis
- Customers with multiple orders
